Installation & upgrade - top

Auto Popups add-on for OptIn Ninja requires WordPress v3.9 or higher to function properly!

Auto Popups add-on for OptIn Ninja requires OptIn Ninja v1.5 or higher to function properly!

Installation

  1. Download the ZIP package from CodeCanyon.
  2. Open WordPress admin and go to Plugins -> Add New -> Upload. Browse for the ZIP file auto-popups-optin-ninja.zip on your computer and click "Install Now".
  3. Activate the plugin.

Upgrade - method #1

  1. Do NOT deactivate the plugin.
  2. Use FTP to locate folder wp-content/plugins/optin-ninja-auto-popups-addon/
  3. Overwrite all files in that folder with new ones downloaded from CodeCanyon

Upgrade - method #2

  1. Deactivate and delete the plugin in your WP admin -> Plugins.
  2. Use the Add New -> Upload function to upload the new plugin ZIP file.
  3. Activate the plugin.

5sec getting started guide - top

Descriptions are available for every option throughout the plugin. Please read them.

Edit one of the existing OptIns and scroll to the bottom of the edit screen to the "Auto Popup / Lightbox" meta box. Enable automatic popup (first option) and scroll down to "Show on Page Load" option, select "Show after 5 seconds". Save OptIn. Refresh any page on your site and after 5 seconds a popup will automatically open.

Settings - top

If the popups are not showing up for you please use the debug tool available in the meta box and remember that all filters are applied to you the same way as they are to other users (ie: number of popup views ...).

All settings related to Auto PopUps are located in the "Auto Popup / Lightbox" meta box. In order to enable auto popups and see all settings you have to enable the first option. If you want to show the A/B Test the Page belongs to instead of page itself, enable the second option.

Second set of options are filters, they determine if the current user is at all eligable to see the popup. You can filter users who have already subscribed, are logged in or logged out (their WP account types don't matter). Last option controls how many times in total a user can see the popup.

Second set of settings are also filters, they control on which pages the popup can be shown. If you leave everything to "disabled" popup will be shown on all pages. If you enable any one of the options they have to be met in order for the popup to show. Custom post types are currently not supported for posts, pages or categories.

Last set of settings are triggers, they determine what action will set of the popup. Of course filters set above have to allow for the popup to get to this stage. First setting opens the popup after page is loaded, second when the user gestures with the mouse that he's about to leave the page, and the last option triggers when a page element you defined comes into view.

FAQ - top

Will this work with my theme?

Yes, it's completely theme independant.

Will popups slow down my site?

Now, all JS is loaded conditionally, only when needed.

How many popups can I create?

As many as you need. Please note that only one popup can be open at a time.
